# POST THIS Design Guide

Deze gids vat de visuele canon samen voor alle developers en designers.

## Designfilosofie
- Rustig, menselijk, licht, tijdloos
- Geen dashboard, generator-tool, marketingplatform, coach-app
- Voelt als een ritueel, een moment, een persoonlijke plek

## Kleuren
- Achtergrond: #F7F7F5 (warm, papiergevoel)
- Tekst: #1E1E1E (zacht donker), #8A8A8A (secundair)
- Accent: #6F7D8C (blauwgrijs), #7A8F7A (groen), #B8AFA6 (zand)

## Typografie
- Inter, SF Pro, system-ui
- Titels: 28–34px, medium
- Vragen: 20–24px
- Body: 16–18px, line-height 1.5–1.6
- Labels: 13–14px, lichtgrijs

## Layout
- Veel witruimte: min. 24px tussen blokken, grote secties 48–72px
- Eén focus per scherm
- Geen visuele druk: geen dikke schaduwen, borders, gradients

## UI Components
- Inputvelden: subtiele border, ruime padding, accentkleur bij focus
- Buttons: accentkleur, afgerond, geen harde hoeken
- Animaties: fade, ease-in-out, 300–600ms

## Fotografie
- Echte mensen, daglicht, candid, geen stock/tech visuals

## Wat mag niet
- Geen SaaS dashboard, generator-UI, Canva-achtige stijl

## Samenvatting
POST THIS ziet eruit als: Daglicht op papier met rustige woorden in een wereld van echte mensen.
